Usage: tacoscript [options] [SUBCOMMAND]

When specified with no arguments, `node` is run in interactive shell mode.

Options:

    --version, -V  Show the version of tacoscript

 --versions, --VV  Show the versions of tacoscript and other related packages

       --help, -h  Show this message


Subcommands:

          compose  Transform tacoscript into javascript

        transpose  Transform javascript into tacoscript

          compile  Transform tacoscript into es5 compatible javascript via babel transforms

             node  Run tacoscript in node, from a file or in an interactive shell
